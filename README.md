# Mental-Health-Awareness-Project

 # Description
Better Mind is a mental health awareness website thats contains calming design, accessible tips, and supportive resources. Built with HTML, CSS, and Bootstrap, the site supports mental wellness through soothing design and positive content.

<a href="assests/images/Mockup.png">Device Mockup</a>

# Features
<ul>
<li>Responsive layout (mobile, tablet, desktop)</li>
<li>Soothing hero section with welcoming message</li>
<li>Informative cards on common mental health challenges</li>
<li>Positive thinking carousel</li>
<li> Mental health resources and helpline contacts</li>
</ul>

# Deployment Procedure

<ol>
